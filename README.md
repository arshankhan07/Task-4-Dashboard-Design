# Sales Financial Dashboard Project

## Objective
Design an interactive dashboard for business stakeholders.

## Tools Used
- Python
- Streamlit
- Plotly
- Pandas
- Python-PPTX

## Features
- KPI Cards (Sales, Profit, Average)
- Time-Series Analysis
- Region & Product Filters
- Top 10 Products Visualization
- Automated PPT Summary Generation

## How to Run Dashboard

1. Install dependencies:
   pip install -r requirements.txt

2. Run Streamlit app:
   streamlit run app.py

3. Generate PPT summary:
   python generate_ppt.py

## Outcome
Helps stakeholders monitor performance, identify trends, and make data-driven decisions.
